# RK Plastics & Enterprises - Invoice Management System (Frontend)

A modern, responsive, and production-ready Invoice Management System built for **RK Plastics & Enterprises**. The application enables businesses to efficiently manage customers, generate GST invoices, download PDF invoices, analyze sales, and monitor business performance through an interactive dashboard.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/pritish9156/rkplastic-invoice-management-frontend.git
```

Go inside project

```bash
cd rkplastic-invoice-management-frontend
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

Preview production

```bash
npm run preview
```
